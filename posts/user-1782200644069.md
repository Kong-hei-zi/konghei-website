✅ 标签筛选

- 文章列表上方新增标签筛选栏（全部 / 随笔 / Web开发 / ...）
- 点击标签即时过滤，只显示匹配文章
- 筛选状态下进入详情、上一篇/下一篇也在筛选范围内

✅ 阅读体验升级

- 阅读时间：每张卡片和详情页都显示 ⏱ X 分钟 估算
- 上一篇/下一篇：详情页底部导航，中间文章两头都有，首篇只有下一篇，末篇只有上一篇
- 排版增强：正文 font-size: 1.02rem + line-height: 2，h2 带底部分隔线，引用块左边框，代码块深色背景

✅ Markdown 编辑器

- 引入 marked.js CDN，支持 Markdown 写作
- 编辑器底部有语法提示（## 标题 **加粗** - 列表 等）
- 兼容旧文章：已有的 HTML 文章会被 marked 透传，正常显示
- 新文章用 Markdown 写，自动渲染

✅ 详情页头部折叠

- 点进文章后，BLOG 博客 记录思考，分享知识 大标题自动收起（max-height: 0; opacity: 0，0.35s 动画）
- 返回列表后自动展开

✅ 卡片视觉升级

- 每张卡片左侧有标签对应颜色的装饰条（随笔=紫，Web开发=青，教程=金...）
- 标签 pill 也同步配色
