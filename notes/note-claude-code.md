## Claude Code 是什么

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，可以直接在终端中帮你写代码、调试、重构。

### 核心功能

- **代码生成**：描述需求，自动生成代码
- **Bug 修复**：粘贴报错，分析并修复
- **代码审查**：检查代码质量和安全问题
- **重构优化**：改进代码结构

### 使用技巧

1. **描述要具体**：`"在 blog.html 中添加标签筛选功能"` 比 `"改一下博客"` 好 10 倍
2. **提供上下文**：告诉它项目结构、技术栈、设计风格
3. **CLAUDE.md**：项目根目录放一个 CLAUDE.md，记录架构和约定
4. **分步推进**：复杂任务拆成小步骤，每步验证

> 把 AI 当成一个**超级实习生**——能力强但需要清晰的指令和上下文！
