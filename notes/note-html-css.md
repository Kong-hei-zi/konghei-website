## HTML 基础

HTML（超文本标记语言）是网页的骨架。每一个网页都由标签组成，浏览器解析这些标签并渲染出我们看到的页面。

### 常用标签

| 标签 | 用途 |
|------|------|
| `<div>` | 块级容器，最常用的布局元素 |
| `<span>` | 行内容器，用于包裹文字 |
| `<h1>~<h6>` | 标题标签，h1 一个页面只用一次 |
| `<p>` | 段落 |
| `<a>` | 超链接 |
| `<img>` | 图片 |

### CSS 核心概念

**盒模型**：每个元素都是一个矩形盒子，由 content → padding → border → margin 四层组成。

**选择器优先级**：`!important` > 内联样式 > ID > class > 标签 > 通配符

**Flexbox**：一维布局利器，`justify-content` 控制主轴，`align-items` 控制交叉轴。

**Grid**：二维布局，`grid-template-columns` 定义列宽，`fr` 单位按比例分配剩余空间。

> 小提示：学会 Flexbox 和 Grid 之后，99% 的布局问题都能解决！
